# 🚀 AB App Store - React Web Application

🔗 **Live Demo:** https://app-store1.netlify.app/

---

## 🌟 Overview

**App Store** is a modern, React-based web application that simulates a digital app marketplace. Users can explore apps, view detailed insights with charts, install or uninstall apps, and manage their installed collection—all within a smooth and responsive interface.

This project focuses on **clean UI, performance, and real-world frontend architecture**, making it a strong demonstration of practical React development skills.

---

## ✨ Key Features

- 🔍 **Explore Apps** – Browse a variety of applications with a clean UI.  
- 📊 **Detailed Insights** – View app details with interactive charts.  
- ⚡ **Install / Uninstall System** – Manage apps dynamically . 
- 💾 **Persistent Storage** – Uses localStorage to track installed apps . 
- 🔎 **Live Search & Sorting** – Quickly find and organize apps . 
- 📱 **Responsive Design** – Works smoothly on mobile & desktop.  
- 🔄 **Smooth Navigation** – Seamless page transitions using React Router .

## ⚙️ How It Works

- Users can browse available apps from the homepage  
- Clicking an app opens a **detailed page with charts & insights**  
- Apps can be **installed or removed instantly**  
- Installed apps are stored in **localStorage**, so data persists even after reload  
- Search and sorting improve usability and user experience
